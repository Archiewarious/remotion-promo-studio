# WELCOME — знакомство с проектом (для Claude)

Это сценарий ПЕРВОГО касания. Когда пользователь только пришёл в воркспейс и ещё не дал
конкретную задачу (приветствие, «что это?», «что умеешь?», «/start», «с чего начать»),
Claude коротко и дружелюбно представляет проект — **на языке пользователя**, без стены текста,
БЕЗ запуска установки. Дальше Claude действует по запросу.

Опорные тезисы (адаптируй, не зачитывай дословно):

**Что это.** Мастерская промо-видео на Remotion v4. Ты описываешь идею/ТЗ — я собираю готовый
ролик: сцены, анимация, тайминг под озвучку, рендер в 4K.

**Что можно (примеры):**
- Сделать видео целиком: «Сделай видео про X. ТЗ: …».
- Превью и рендер, субтитры, музыка + голос (с дакингом), реформат под Reels/Shorts/квадрат,
  поиск b-roll со стоков.
- Команды (если интересно): `/render`, `/studio`, `/sheet`, `/mix`, `/reframe`, `/broll`, `/timing`.

**Доп. ключи (опционально).** Нужны только для авто-поиска b-roll (`/broll`) — бесплатные:
- Pexels — https://www.pexels.com/api/
- Pixabay — https://pixabay.com/api/docs/
Получил ключ → впиши в файл `.env` (скопируй из `.env.example`, формат `PEXELS_API_KEY=...`).
Без ключей всё остальное работает — стоки можно класть в `public/` вручную.

**Как сформулировать запрос на видео (чем конкретнее — тем лучше):**
> «Сделай 15-сек промо для кофейни «Аура». Тон тёплый, уютный. Лого — в `public/logo.png`,
>  цвета бежевый/коричневый. Музыка спокойная. Финал — призыв "Заходи на чашку".»
Полезно указать: длительность, площадку (YouTube/Reels), тон, цвета/лого, текст/слоган.

(зависимости, ffmpeg) и как собрать ролик.
