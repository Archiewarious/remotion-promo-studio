---
description: Remotion Studio (живой превью, hot-reload) — самый быстрый цикл правок
argument-hint: (запускать из папки проекта)
---
Запусти живой превью В ФОНЕ из папки проекта: `npx remotion studio src/index.ts --gl=angle-egl` → http://localhost:3000. Правишь `.tsx` — обновляется мгновенно (быстрее, чем гонять рендеры). Это основной инструмент итераций; покадровая vision-оценка — `/still` и `/render preview` + `/sheet`.
